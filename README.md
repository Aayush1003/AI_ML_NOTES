# 🧠 AI & ML Comprehensive Study Notes

Welcome to the **AI & ML Notes** repository! This repository is designed to be a comprehensive, structured guide for mastering Artificial Intelligence, Machine Learning, Deep Learning, and related fields. It serves as a central hub for learning, revision, and interview preparation.

---

## 📂 Repository Structure

### [01. Prerequisites](./01_Prerequisites/)
Foundation required before diving into ML models.
*   **Mathematics:** Linear Algebra, Calculus, Probability & Statistics.
*   **Programming:** Python for Data Science (NumPy, Pandas, Matplotlib).

### [02. Machine Learning](./02_Machine_Learning/)
Core concepts of traditional Machine Learning algorithms.
*   **Supervised Learning:** Linear/Logistic Regression, SVMs, Decision Trees.
*   **Unsupervised Learning:** K-Means, Hierarchical Clustering, PCA.
*   **Ensemble Methods:** Random Forests, Gradient Boosting (XGBoost, LightGBM).
*   **Model Evaluation:** Bias-Variance Tradeoff, Cross-Validation, Metrics (Accuracy, F1, ROC-AUC).

### [03. Deep Learning](./03_Deep_Learning/)
Modern neural networks and architectures.
*   **Foundations:** Perceptrons, Activation Functions, Backpropagation, Optimizers.
*   **Computer Vision:** Convolutional Neural Networks (CNNs), ResNet, Object Detection.
*   **Sequence Models:** RNNs, LSTMs, GRUs.
*   **Transformers:** Attention Mechanisms, Vision Transformers.

### [04. Natural Language Processing (NLP)](./04_Natural_Language_Processing/)
Working with text data.
*   **Text Processing:** Tokenization, Stemming, Lemmatization, TF-IDF.
*   **Word Embeddings:** Word2Vec, GloVe, FastText.
*   **Modern NLP:** BERT, GPT, Sequence-to-Sequence models.

### [05. Generative AI](./05_Generative_AI/)
The cutting edge of AI.
*   **Large Language Models (LLMs):** Architecture, Fine-tuning, Prompt Engineering.
*   **RAG (Retrieval-Augmented Generation):** Vector Databases, LangChain, LlamaIndex.
*   **Advanced Agentic Workflows & MCP:** Model Context Protocol (MCP), LangGraph state machines, multi-agent systems.
*   **Image Generation:** GANs, Diffusion Models (Stable Diffusion, Midjourney).

### [06. MLOps](./06_MLOps/)
Taking models from notebooks to production.
*   **Deployment:** Flask/FastAPI, Docker, Kubernetes.
*   **Monitoring & Tracking:** MLflow, Model Drift detection.
*   **Cloud ML:** Azure ML / AWS SageMaker basics.

### [07. Interview Prep](./07_Interview_Prep/)
*   ML System Design (e.g., Designing a Recommendation System).
*   Top Conceptual Questions & Coding patterns.
*   Cheat sheets and quick revision guides.
