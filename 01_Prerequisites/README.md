# Prerequisites for Machine Learning & AI

Before diving into complex machine learning algorithms, it is crucial to have a strong foundation in Mathematics and Programming. 

## 1. Mathematics
Machine learning is essentially applied math. Understanding what happens under the hood requires grasping these core domains:

### Linear Algebra
*   **Concepts:** Scalars, Vectors, Matrices, Tensors.
*   **Operations:** Matrix Multiplication, Inverse, Transpose.
*   **Advanced:** Eigenvalues and Eigenvectors, SVD (Singular Value Decomposition), PCA (Principal Component Analysis).
*   **Why?** Used heavily in deep learning (neural network weights are matrices), dimensionality reduction, and image processing.

### Calculus
*   **Concepts:** Derivatives, Partial Derivatives, Chain Rule.
*   **Optimization:** Gradient Descent (understanding how loss functions are minimized to update model weights).
*   **Why?** Essential for backpropagation in neural networks and optimizing any ML model.

### Probability & Statistics
*   **Probability:** Bayes' Theorem, Random Variables, Probability Distributions (Normal, Binomial, Poisson).
*   **Statistics:** Mean, Median, Variance, Standard Deviation, Hypothesis Testing, p-values.
*   **Why?** Used in Naive Bayes, Hidden Markov Models, understanding data distributions, and evaluating models (confidence intervals).

## 2. Programming (Python)
Python is the industry standard for AI/ML due to its readable syntax and massive ecosystem.

### Core Libraries
*   **NumPy:** High-performance multi-dimensional array and matrix operations (Linear Algebra in code).
*   **Pandas:** Data manipulation and analysis (DataFrames, cleaning, filtering).
*   **Matplotlib / Seaborn:** Data visualization (plotting distributions, correlations, loss curves).
*   **Scikit-Learn:** The go-to library for traditional machine learning algorithms and preprocessing tools.

## Recommended Study Path
1. Master Python syntax and structures (Lists, Dicts, Loops).
2. Learn NumPy and Pandas for data manipulation.
3. Review foundational Math concepts as they appear (e.g., learn derivatives right before studying Gradient Descent).
